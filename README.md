# chtech-website-v2

誠創科技網站 V2 Firebase 測試版。

## 目標

GitHub Pages 只放網站外殼，文字資料與圖片未來全部放 Firebase。

## 不放在 GitHub 的內容

- 公司資料
- 客戶案例圖片
- 輪播圖片
- 最新消息圖片
- 表單資料
- 後台帳號密碼

## 上線流程

1. 上傳此資料夾到 GitHub Repo：chtech-website-v2
2. GitHub Pages 設定 main / root
3. 建立 Firebase 專案
4. 將 Firebase config 貼入 assets/js/firebase-config.js
5. 後續再開啟 Firestore / Storage / Auth 串接
