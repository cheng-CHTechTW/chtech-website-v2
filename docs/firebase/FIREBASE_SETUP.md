# Firebase 設定步驟

1. 到 Firebase Console 建立專案：CHTech-V2
2. 建立 Web App
3. 開啟 Authentication：Email/Password
4. 建立 Firestore Database
5. 建立 Storage
6. 將 Firebase Web App config 貼到 assets/js/firebase-config.js
7. 把 firebaseEnabled 改成 true

## Firestore Collections

- siteSettings/main：公司基本資料
- banners：首頁輪播
- services：服務項目
- cases：客戶案例
- news：最新消息
- faq：常見問題
- forms：表單紀錄
- users：後台權限

## Storage Buckets 路徑

- company/
